Hey, I'm Andrew! I'm a manufacturing engineer transitioning into computer science.
I studied Engineering Product Design at university, but now I’m diving into software development.

This website is part of one of the problem sets from Harvard’s CS50: Introduction to Computer Science course. It’s a practice project for writing HTML, CSS, and JavaScript, and for implementing features from a library (in this case, Bootstrap). I decided to go for a dark-mode design with black-and-white images. This site gives a short introduction about me — where I’m from, my studies, and my interests.

There are four HTML pages: index.html, my-background.html, my-education.html, and my-passion.html. You can navigate between them using the top menu.

Besides the basic tags html, head, body, and title, this site also uses: link, div, nav, ul, li, br, button, a, p, code, and others.

This accordion itself is built with Bootstrap! The top navigation menu and the three-column layout below are also Bootstrap components.

This site uses my own stylesheet (styles.css) and includes several CSS selectors, such as: body, .main-titles, img, .main-text, .col, button, etc. It also uses multiple CSS properties, such as: color, text-align, font-family, font-size, margin-left, margin-right, padding-left, and padding-right.

JavaScript is used for the Welcome button on the homepage, which asks for the user’s name and displays “Hello, NAME. Welcome to my website!” It’s also used for the typing effect on the main titles of each page, which runs in a loop.

All items are placed inside flex containers, so when the display size changes, their spacing and positioning automatically adjust. On mobile devices, a hamburger icon appears to access the top menu.
